# Code Review Skill

## Description
Expert code review capabilities with focus on best practices, security, and maintainability.

## Capabilities
- Static analysis and code quality assessment
- Security vulnerability detection
- Performance optimization suggestions
- Code style and convention enforcement
- Documentation quality evaluation

## Usage
When reviewing code, I will:
1. Check for common vulnerabilities and security issues
2. Evaluate code structure and design patterns
3. Suggest improvements for readability and maintainability
4. Verify proper error handling and edge cases
5. Assess test coverage and quality

## Best Practices
- Follow language-specific conventions
- Prioritize security and performance
- Provide constructive, actionable feedback
- Consider team coding standards
- Balance thoroughness with practicality
